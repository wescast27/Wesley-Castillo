The Prompt:

Your instructor decides to quit DU and move to Japan to be with his daughter and new grandchild. Since Japan is like England and the driver sits on the right and traffic flows on the left side of the road, his American cars are useless in Japan. Therefore, he is going to give his 5 cars (Mazda CX-5, Tesla Model 3, Ford Focus, Tesla Cybertruck, and Jeep Wrangler), to his three favorite INFO 3440 students. You know who you are :) but for these purposes, we will call you Student 1, Student 2, and Student 3.

The Constraints

  • No student can receive more than three cars

  • Each student must receive at least one car

  • Your instructor doesn't want to deal with this anymore, so all the cars must be given away

  • If a student gets one of the Teslas, (s)he cannot receive the other Tesla

  • Student 1 is obviously his most favorite, so in this case Student 2 may not receive more cars than student 1

  • Obviously, the same car can not be given to more than 1 student

  • To help with gas mileage concerns, If the Jeep Wrangler goes to Student 1, then Student 1 must also get the Ford Focus.

  • If Student 1 gets the Ford Focus OR if Student 2 gets the Cybertruck, then the CX-5 must go to Student 3.

  • As we have also established, your instructor is basically lazy and wants to minimize the number of miles he has to drive to drop these cars off at the students' houses. Student 1 lives 17 miles away from your instructor, where student 2 lives 12 miles away and student 3 lives 15 miles away

Model Creation:

model = pe.ConcreteModel()
cars = ['Mazda CX-5', 'Tesla Model 3', 'Ford Focus', 'Tesla Cybertruck', 'Jeep Wrangler']
students = ['S1', 'S2', 'S3']
miles = {'S1': 17, 'S2': 12, 'S3': 15}
model.x = pe.Var(students, cars, within=pe.Binary)
# Binary decision variable for car assignment (1 if car i is given to student j, 0 otherwise)
model.assign = Var(((student, car) for student in students for car in cars), within=Binary)

Above is another thing I got from the Problem 2 AI file. This binary decision variable is used in the other file to help create all constraints, however rather than chang all my constraints which worked fine, I am just using this to cut down the code needed for the final cx-5 constraint.

def objective_rule(model):
    return sum(model.x[student, car] * miles[student] for student in students for car in cars)
model.objective = pe.Objective(rule=objective_rule, sense=pe.minimize)

Constraints:

#there is only one of each car to give
def one_of_each(model, car):
    return sum(model.x[student, car] for student in students) == 1
model.one_car_per_student_constraint = pe.Constraint(cars, rule=one_of_each)

#No student can have more than 3 cars
def max_cars_3(model, student):
    return sum(model.x[student, car] for car in cars) <= 3
model.max_cars_constraint = pe.Constraint(students, rule=max_cars_3)

#Every student must recieve at least 1 car
def min_cars_1(model, student):
    return sum(model.x[student, car] for car in cars) >= 1
model.min_cars_constraint = pe.Constraint(students, rule=min_cars_1)

#Nobody can recieve both tesla cars
def tesla_1_only(model, student):
    return model.x[student, 'Tesla Model 3'] + model.x[student, 'Tesla Cybertruck'] <= 1
model.tesla_constraint = pe.Constraint(students, rule=tesla_1_only)

#Student 2 can't recieve more cars than student 1
def favorite_student(model):
    return sum(model.x['S2', car] for car in cars) <= sum(model.x['S1', car] for car in cars)
model.student_priority_constraint = pe.Constraint(rule=favorite_student)

#If the jeep goes to student 1 then he must also get the ford focus
def jeep_with_focus(model):
    return model.x['S1', 'Jeep Wrangler'] - model.x['S1', 'Ford Focus'] <= 0
model.jeep_focus_constraint = pe.Constraint(rule=jeep_with_focus)

#If student 1 gets the focus or if student 2 gets the cybertruck then the cx-5 goes to student 3
def cx5_rule(model): #In my original file I split this constraint into 2, but using the model.assign I condensed it.
    return model.assign['S1', 'Ford Focus'] + model.assign['S2', 'Tesla Cybertruck'] - model.assign['S3', 'Mazda CX-5'] <= 0
model.cx5_constraint = pe.Constraint(rule=cx5_rule)
